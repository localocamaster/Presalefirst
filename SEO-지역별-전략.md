# 분양퍼스트 지역별 SEO 전략 문서

## 1. 전략 개요

### 참고 사례: 센스드라이브 (운전연수 사이트)
- **URL 패턴**: `/driving-lessons/{지역명}` (예: `/driving-lessons/seoul`, `/driving-lessons/gangnam`)
- **핵심 전략**: 지역명 + 타겟 키워드("운전연수")를 조합하여 각 지역별 독립 페이지를 생성
- **효과**: 지역 검색 트래픽을 각 페이지로 분산 획득 (롱테일 키워드 전략)

### 분양퍼스트 적용 방향
- **URL 패턴**: `/map/{지역slug}` (예: `/map/seoul`, `/map/seoul-gangnam`, `/map/gyeonggi-suwon`)
- **타겟 키워드**: "분양홈페이지 제작", "분양웹사이트 제작", "분양 홈페이지"
- **조합 예시**: "서울 분양홈페이지 제작", "강남 분양홈페이지 제작", "수원 분양홈페이지 제작"
- **전략**: 광역시/도 단위 페이지 + 모든 구/시/군 단위 페이지를 전부 생성하여 롱테일 키워드 완전 장악
- **총 페이지 수**: 광역 12개 + 하위 지역 약 130개 = **약 142개 지역 페이지**

## 3. 페이지 구조 (온페이지 SEO)

### 메타 태그 패턴 (2026 SEO 최적화)

**메타 타이틀 원칙:**
- 55자 이내 (한글 기준 ~25자)
- 키워드를 제목 앞에 배치 (keyword-first)
- USP(가격, 시간 등) 포함
- 브랜드명은 마지막에 `| 분양퍼스트`
- SeoHead 컴포넌트에서 `exactTitle` prop 사용 (자동 래핑 방지)

```
<title>{지역} 분양홈페이지 제작 | 19만원 24시간 완성 | 분양퍼스트</title>
```

**메타 설명 원칙:**
- 120~160자 (잘리지 않는 범위)
- 지역명 + 하위지역명 포함 (롱테일 키워드)
- 가격·시간 등 구체적 USP 포함
- 마지막에 CTA 포함 (예: "~까지")

```
<meta name="description" content="{지역} 분양상담사 전용 홈페이지 제작. {하위지역1}, {하위지역2}, {하위지역3}, {하위지역4} 등 {지역} 전 지역 대응. 19만원부터, 24시간 내 제작 완료. 전화·카톡 상담, 관심고객 관리." />
```

**서울 예시:**
```
title: 서울 분양홈페이지 제작 | 19만원 24시간 완성 | 분양퍼스트
description: 서울 분양상담사 전용 홈페이지 제작. 강남, 송파, 마포, 서초 등 서울 전 지역 대응. 19만원부터, 24시간 내 제작 완료. 전화·카톡 상담, 관심고객 관리.
```

### H 태그 구조 (공감형 카피라이팅)

H 태그는 단순 키워드 나열이 아닌, 감정적 공감 → 문제 인식 → 해결책 → CTA 흐름으로 구성합니다.

```
H1: {지역} 분양홈페이지 제작
  H2: {지역}에서 분양하면서, 홈페이지 없이 버티고 계신가요?  ← 공감형 도입
  H2: 외주는 비싸고, 직접 만들기는 막막하셨죠               ← 현실적 고민 공감
  ── [제작 과정 3단계 비주얼 섹션] ──
  H2: {지역} 분양상담사를 위한 홈페이지, 19만원이면 충분합니다 ← 해결책 제시
    H3: {지역} 전 지역 분양현장에 대응합니다
    H3: 분양에 꼭 필요한 기능만 담았습니다
  ── [기능 그리드 비주얼 섹션] ──
  H2: {지역} 분양시장, 왜 홈페이지가 더 중요한가             ← 시장 특화 설득
  H2: 네이버 파워링크 광고, 홈페이지가 있어야 시작할 수 있습니다
  ── [중간 CTA + 요금제 카드 비주얼 섹션] ──
  ── [FAQ 섹션] ──
  ── [최종 CTA 섹션] ──
```

**중요: H2 중복 방지**
- "제작 비용" H2는 PricingSection 컴포넌트에서만 렌더링 (content 데이터에 넣지 않음)
- "지금 시작하세요" H2는 최종 CTA 섹션에서만 렌더링 (content 데이터에 넣지 않음)
- 비주얼 섹션이 자체 H2를 가지므로 content 데이터와 중복되지 않도록 주의

### 콘텐츠 + 비주얼 섹션 교차 배치 (break 시스템)

content 배열에 `{ type: 'break', text: '{섹션명}' }` 마커를 삽입하면, 해당 위치에 비주얼 섹션이 삽입됩니다.

| break text | 삽입되는 섹션 | 설명 |
|------------|--------------|------|
| `steps` | StepsSection | 제작 과정 3단계 카드 |
| `features` | FeaturesSection | 포함 기능 8개 그리드 |
| `pricing` | MidCta + PricingSection | 중간 CTA 배너 + 3개 요금제 카드 |

**배치 원칙:**
- 글만 계속 이어지지 않도록 2~3개 텍스트 섹션마다 비주얼 섹션 삽입
- 상단에도 비주얼을 배치하여 시각적 밸런스 유지
- Hero → 텍스트 → Steps → 텍스트 → Features → 텍스트 → CTA+Pricing → 텍스트 → FAQ → 최종 CTA

### 카피라이팅 원칙 (2sama 블로그 참고)

1. **공감형 도입**: "~하고 계신가요?", "~하셨죠" 형태로 고객의 고민을 먼저 짚음
2. **문제 → 공감 → 해결 → CTA** 흐름 유지
3. **구체적 수치 제시**: "300만 원~500만 원", "19만원", "24시간", "10명 중 8명"
4. **고객의 언어 사용**: "조금 더 자리 잡히면 만들죠" 같은 실제 고객 심리 반영
5. **지역 고유 정보 포함**: 하위 지역명, 시장 특성, 개발 호재 등

### 콘텐츠 구성 원칙
1. **자연스러운 키워드 배치**: 제목, 본문 첫 문단, 소제목에 "지역명 + 분양홈페이지 제작" 키워드 배치
2. **지역 고유 정보 포함**: 해당 지역의 분양시장 특성, 개발 호재, 교통 정보 등 고유한 내용 포함
3. **내부 링크**: 다른 지역 페이지, 블로그, 서비스 페이지, 문의 페이지로의 내부 링크 포함
4. **FAQ 스키마**: 각 지역별 자주 묻는 질문을 FAQ 구조화 데이터로 마크업
5. **70% 이상 고유 콘텐츠**: 지역별 페이지 간 중복 콘텐츠 방지

## 4. 구조화 데이터 (JSON-LD Schema)

각 지역 페이지에 포함되는 스키마 6종:
1. **WebPage**: 페이지 기본 정보 (name, description, breadcrumb 연결)
2. **BreadcrumbList**: 홈 > 지역별 제작 > {지역} 분양홈페이지 제작
3. **FAQPage**: 지역별 자주 묻는 질문 (리치 스니펫 노출 대상)
4. **Product**: 3개 요금제 정보 (name, price, description) → 가격 리치 스니펫
5. **ProfessionalService (LocalBusiness)**: 사업자 정보, 주소, 좌표, 서비스 지역, 가격대
6. **Service**: 서비스 유형, 제공자 정보, 3개 Offer (베이직/디럭스/프리미엄)

스키마 생성 유틸: `src/utils/schema.ts`
- `getWebPageSchema()`, `getBreadcrumbSchema()`, `getFAQPageSchema()`
- `getProductSchema()`, `getLocalBusinessSchema()`, `getServiceSchema()`

## 6. 콘텐츠 작성 가이드라인

### 카피라이팅 흐름 (공감형, 2sama 블로그 참고)
```
1. 공감형 도입: 고객의 현재 상황과 고민을 짚어줌
2. 현실적 문제: 비용 부담, 시간 부족 등 공감
3. 해결책 제시: 분양퍼스트 서비스 소개 (가격, 속도)
4. 지역 특화 정보: 하위 지역, 시장 현황, 고유 콘텐츠
5. 파워링크 연동: 광고 활용 가치 설명
6. 추가 가치: 접속통계 등 차별점
→ FAQ → 최종 CTA
```

### 지역별 고유 콘텐츠 작성 시 포함할 내용
1. **지역 부동산 시장 현황**: 최근 분양 트렌드, 주요 개발 사업, 경쟁 강도
2. **대표 분양 지역**: 해당 지역 내 인기 분양 지역 (구, 동 단위) — 하위지역 최소 4개 이상 나열
3. **교통 호재**: GTX, 지하철, 고속도로 등 교통 인프라 (지역 고유 정보)
4. **고객 검색 행동**: "고객 10명 중 8명은 상담 전에 먼저 검색합니다" 같은 설득 포인트
5. **분양퍼스트 서비스 연결**: 자연스럽게 가격·속도·기능 소개 → CTA로 연결

### 주의사항
- 지역별 페이지의 콘텐츠가 **70% 이상 고유**해야 함 (중복 콘텐츠 방지)
- 키워드 밀도는 본문 대비 2~3% 유지
- 최소 **800자 이상**의 본문 작성 (서울 템플릿은 약 2,000자)
- 이미지에 alt 태그로 "지역명 + 분양홈페이지" 포함
- content 데이터에 **break 마커 3개** 필수 배치 (steps → features → pricing)
- PricingSection, 최종 CTA가 자동으로 H2를 렌더링하므로 **content에 해당 H2를 넣지 말 것**
